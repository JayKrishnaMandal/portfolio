# JKChat - Realtime Messenger

A modern, mobile-responsive chat application built with Vanilla JS and Firebase Realtime Database.

## Features

- 📱 **Fully Responsive**: Works perfectly on Desktop and Mobile (Android/iOS).
- 💬 **Realtime Messaging**: Instant delivery with online/offline status.
- 🔒 **Secure**: Room-based authentication with private keys.
- 🎨 **Premium UI**: Dark mode, gradients, and glassmorphism.
- 😊 **Emoji Picker**: Integrated support for emojis.

## Hosting on GitHub Pages

1.  Push this repository to GitHub.
2.  Go to Settings -> Pages.
3.  Select `Source: Deploy from branch` -> `master` (or `main`) -> `/ (root)`.
4.  Save.

## Configuration

This app uses Firebase for the backend. The configuration is in `firebase-config.js`.
